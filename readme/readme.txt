Plugin for CudaText.
Gives commands to open CudaText API wiki as file and jump to specified section.

#### Commands

`Open wiki file` - just opens API wiki in CudaText.

`Jump to section` - shows a dialog menu where the user can choose the section of the API.

#### Online mode:
	
	If the user deletes or renames local wiki file in the plugin directory, plugin will download content of the wiki file from github directly every time.
